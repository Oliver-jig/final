# 🐱 Ding Ding Cat Sticker Generator

AI-powered festival sticker generator featuring Ding Ding Cat!

## Setup

1. `npm install`
2. Copy `.env.example` to `.env.local` and fill in your keys
3. `npm run dev`

## Deploy to Vercel

1. Push to GitHub (never commit `.env.local`)
2. Import on vercel.com
3. Add env vars in Vercel dashboard → Settings → Environment Variables:
   - `AI_GATEWAY_API_KEY`
   - `NOTION_TOKEN`
4. Deploy!
